# 🏦 Bank Customer Churn Prediction

## 📌 Project Overview
Predict customer churn using Machine Learning and visualize insights using Streamlit.

## 🚀 Features
- Churn prediction with probability
- Risk classification (Low/Medium/High)
- Feature importance visualization
- What-if analysis

## 🛠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas, NumPy

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
